# WURP — Toxic Motion Engine


WURP is a toxic gas themed hybrid synth/effect plugin.

It mutates audio using formant motion, corrosive saturation,
elastic pitch movement, and animated reactive UI themes.

Designed to create:

• rubber bass
• talking synths
• bubbling plucks
• mutant FX
• toxic atmospheres

---

## Features

• Hybrid instrument + FX mode
• Gas generator noise engine
• Toxic filter models
• Formant mouth morphing
• Corrosive saturation
• Motion sequencer
• Groove modulation
• Animated UI themes
• Custom theme packs
• Audio reactive visuals

## Themes

WURP includes animated toxic themes and supports custom theme packs.

Supported formats:

• GIF
• animated WebP
• APNG
• sprite sheets

Themes can include custom particles, overlays, and color palettes.

# WURP Product Line Package

This package contains:

- `WURP/` — the $7 open GitHub-friendly version
- `WURP_Toxicron_Edition/` — the $13 premium closed-source direction scaffold

This handoff is more complete than the earlier minimal scaffold and includes:
- CMake/JUCE-oriented project structure
- processor/editor skeletons
- DSP module stubs
- theme system schema + examples
- preset examples
- docs/specs/build notes
- validation/test placeholders
- scripts for packaging and theme checks

Important:
This is still a development handoff package, not a compiled retail plugin binary.
