# WURP Product Line Package

This package contains:

- `WURP/` — the $7 open GitHub-friendly version
- `WURP_Toxicron_Edition/` — the $13 premium closed-source direction scaffold

This handoff is more complete than the earlier minimal scaffold and includes:
- CMake/JUCE-oriented project structure
- processor/editor skeletons
- DSP module stubs
- theme system schema + examples
- preset examples
- docs/specs/build notes
- validation/test placeholders
- scripts for packaging and theme checks

Important:
This is still a development handoff package, not a compiled retail plugin binary.
