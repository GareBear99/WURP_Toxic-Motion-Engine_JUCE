# WURP — Toxic Motion Engine

WURP is a toxic gas themed hybrid synth/effect plugin.

It mutates audio using formant motion, corrosive saturation,
elastic pitch movement, and animated reactive UI themes.

Designed to create:

• rubber bass
• talking synths
• bubbling plucks
• mutant FX
• toxic atmospheres

---

## Features

• Hybrid instrument + FX mode
• Gas generator noise engine
• Toxic filter models
• Formant mouth morphing
• Corrosive saturation
• Motion sequencer
• Groove modulation
• Animated UI themes
• Custom theme packs
• Audio reactive visuals

---

## Themes

WURP includes animated toxic themes and supports custom theme packs.

Supported formats:

• GIF
• animated WebP
• APNG
• sprite sheets

Themes can include custom particles, overlays, and color palettes.
